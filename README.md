# git-cheat

> Beautiful, easy to reach Git cheat sheet in your terminal

![git cheat](https://imgur.com/WKhXa54.gif)

## Usage

```sh
$ git cheat 
```

## Installation

##### Using NPM

```sh
sudo npm install --global @fakerr/gitchsh
```
## Requirements
- Tools: git
